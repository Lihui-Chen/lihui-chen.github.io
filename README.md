# 个人学术主页（单页）

这是一个极简的一页学术主页模板，维护简单：只需编辑 `content.md` 即可更新页面内容。可直接部署到 GitHub Pages（无需构建）。

快速使用：

1. 在 GitHub 上创建一个新仓库（例如 `yourname.github.io` 或任意仓库）。
2. 将本项目文件推送到仓库的 `main`（或 `master`）分支。
3. 在仓库设置中启用 GitHub Pages：
   - 打开 Settings → Pages → Source，选择 `main` 分支和 `/ (root)` 目录，然后保存。
4. 等待几分钟，页面将通过 `https://<你的用户名>.github.io/<仓库名>/`（或 `https://<你的用户名>.github.io/` 若仓库为 `yourname.github.io`）访问。

维护：

- 编辑 `content.md`，提交并推送到仓库，GitHub Pages 会自动发布更新。
- 若要添加图片或 CV，把它们放在仓库中并在 `content.md` 用相对路径引用（例如 `images/cv.pdf`）。

备注：如果你使用 `yourname.github.io` 仓库并将文件放在根目录，访问地址会是 `https://yourname.github.io/`。
